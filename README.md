# rbi.js
rbi.js let's you control background images responsively!

Example

```html
<div class="rbi"
	data-rbi-sm=""
	data-rbi-md=""
	data-rbi-lg=""
	data-rbi-xlg=""
	data-rbi-xxlg=""

	data-rbi-sm2x=""
	data-rbi-md2x=""
	data-rbi-lg2x=""
	data-rbi-xlg2x=""
	data-rbi-xxlg2x=""

	data-rbi-size=""
	data-rbi-position=""
	data-rbi-attachment=""
></div>
```
